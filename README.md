# 🍔 Food Delivery Application

A dynamic **Food Delivery Web Application** developed using **Java, JSP, Servlets, JDBC, and MySQL**. The application allows users to register, log in, browse menus, manage carts, and place food orders through an interactive interface.

## 🚀 Features

- User Registration & Login Authentication
- Session Management
- Restaurant/Menu Display
- Add to Cart Functionality
- Cart Management
- Order Placement & Checkout
- Database Connectivity using MySQL
- Responsive User Interface

## 🛠️ Technologies Used

- Java
- JSP & Servlets
- JDBC
- MySQL
- HTML
- CSS
- Apache Tomcat
- Eclipse IDE
- Git & GitHub
